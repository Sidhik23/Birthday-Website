# 🎂 Birthday Website

A responsive and interactive birthday-themed web application built using HTML, CSS, and JavaScript.  
This project features animations, music integration, and multi-step UI interactions to create an engaging user experience.

---

## ✨ Features

* 🎉 Animated countdown and transition effects
* 🎂 Interactive cake cutting experience
* 🎈 Dynamic decorations (balloons, banners, candles)
* 💖 Smooth page transitions and romantic-themed UI
* 🖼️ Image gallery with carousel
* 🎵 Background music integration
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Animations, Flexbox, Glassmorphism)
* JavaScript (DOM Manipulation, Events)
